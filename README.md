# Algorithmic-Log-Analyzer-Python

This project analyzes server log files to extract useful summary metrics and detect patterns in system activity.

## Features
- Processes large text-based log files
- Counts message types (ERROR, INFO, WARNING)
- Tracks IP address activity
- Identifies the most active IP address
- Extracts unique log messages
- Uses dictionaries, lists, and sets for efficient aggregation
